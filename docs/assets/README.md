# App Store Assets

This directory contains visual assets required for App Store submission.

## App Icon

Place your 1024x1024px app icon here:
- **Filename**: `app-icon.png`
- **Size**: 1024 x 1024 pixels
- **Format**: PNG (no transparency)
- **Color Profile**: sRGB or Display P3
- **Design**: Should reflect HALO's dark-first, glassmorphism aesthetic

### Design Guidelines for App Icon
- Use Void Black background or dark gradient
- Incorporate glassmorphism if possible at this scale
- Rounded corners will be applied automatically by iOS
- No text (app name is shown separately)
- Should be recognizable at small sizes
- Must be unique and not confuse with other apps
