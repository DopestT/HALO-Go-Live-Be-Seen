### Description

HALO is a premium live streaming platform where authenticity meets artistry. Go live and be seen in your most genuine moments.

**Key Features:**
- High-quality live streaming
- Real-time interaction with viewers
- Discover trending streams
- Safe and moderated community
- Adult mode for 18+ content (with age verification)

**Design Philosophy:**
- Dark-first aesthetic with glassmorphism
- Presence over performance
- Calm, respectful user experience
- No gaming HUDs or urgency-based language

### Privacy & Safety

- Age verification for adult content
- One-tap reporting system
- User blocking capabilities
- Content moderation
- No data selling or tracking
