# Screenshots Directory

This directory contains App Store screenshots for various device sizes.

## Directory Structure

```
screenshots/
├── iphone-6.7/     # iPhone 14/15 Pro Max (1290 x 2796 px)
├── iphone-6.5/     # iPhone 11/XS Max (1242 x 2688 px)
└── ipad-12.9/      # iPad Pro 12.9" (2048 x 2732 px)
```

## Naming Convention

Use descriptive names that indicate the order and content:

```
01-onboarding.png
02-login.png
03-discovery.png
04-livestream.png
05-profile.png
```

## Screenshot Requirements

### iPhone 6.7" (Primary)
- **Size**: 1290 x 2796 pixels
- **Count**: 3-10 screenshots
- **Required**: Yes (minimum requirement)

### iPhone 6.5"
- **Size**: 1242 x 2688 pixels  
- **Count**: 3-10 screenshots
- **Required**: Yes (for older devices)

### iPad 12.9"
- **Size**: 2048 x 2732 pixels
- **Count**: 3-10 screenshots
- **Required**: Only if supporting iPad

## Screenshot Checklist

For each device size, include:
- [ ] Onboarding/splash screen
- [ ] Login screen
- [ ] Discovery feed
- [ ] Live stream view
- [ ] Profile/settings screen

## Guidelines

- All screenshots must be actual app screenshots (no mockups)
- Maintain HALO's dark aesthetic and glassmorphism design
- No placeholder or test content
- High quality (PNG preferred)
- Portrait orientation only

See `../SCREENSHOT_GUIDELINES.md` for detailed instructions.
