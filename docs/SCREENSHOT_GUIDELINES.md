# Screenshot Guidelines for App Store Submission

## Required Screenshot Sizes

### iPhone 6.7" Display (iPhone 14 Pro Max, 15 Pro Max)
**Size**: 1290 x 2796 pixels
**Format**: PNG or JPEG
**Quantity**: 3-10 screenshots

### iPhone 6.5" Display (iPhone 11 Pro Max, XS Max)
**Size**: 1242 x 2688 pixels  
**Format**: PNG or JPEG
**Quantity**: 3-10 screenshots

### iPad Pro (12.9-inch, 6th generation)
**Size**: 2048 x 2732 pixels
**Format**: PNG or JPEG
**Quantity**: 3-10 screenshots (if supporting iPad)

## Screenshot Sequence

Present screenshots in this order to tell the HALO story:

### 1. Onboarding/Splash Screen
**Purpose**: Show first impression and brand identity
**Key Elements**:
- HALO logo
- Glassmorphism design
- Dark aesthetic
- Tagline: "Go Live, Be Seen"

### 2. Login/Registration Screen
**Purpose**: Show clean, accessible authentication
**Key Elements**:
- Glassmorphism card
- Simple input fields
- No urgency-based copy
- "Create Account" and "Sign In" options

### 3. Discovery Feed
**Purpose**: Showcase content browsing experience
**Key Elements**:
- Low-density layout
- Live stream thumbnails
- Viewer counts (using tabular numbers)
- Generous spacing between items
- No overwhelming UI elements

### 4. Live Stream View
**Purpose**: Display the core streaming experience
**Key Elements**:
- Full-screen video
- Clean chat interface
- Minimal, unobtrusive controls
- Engagement metrics
- Safety features visible (Report/Block)

### 5. Profile/Settings Screen
**Purpose**: Show personalization and safety features
**Key Elements**:
- User profile customization
- Adult Mode toggle
- Privacy settings
- Safety features (Report, Block)
- Clean, organized layout

## Design Guidelines

### Do's
✅ Use actual app screenshots (no mockups)
✅ Show real content (or realistic placeholder content)
✅ Maintain dark theme throughout
✅ Highlight glassmorphism effects
✅ Show rounded geometry (radius: 24)
✅ Use high-quality, crisp images
✅ Demonstrate key features
✅ Show safety features prominently
✅ Include descriptive text overlays (optional)

### Don'ts
❌ No placeholder lorem ipsum text
❌ No obvious debug/test content
❌ No neon colors or gaming aesthetics
❌ No urgency-based marketing copy
❌ No adult/NSFW content
❌ No fake metrics or engagement numbers
❌ No competitor branding
❌ No watermarks

## Text Overlays (Optional)

**Position**: Top or bottom, not covering key UI elements

## Technical Requirements

### File Format
- **Preferred**: PNG (for better quality)
- **Acceptable**: JPEG (95%+ quality)
- **Color Profile**: sRGB or Display P3

### File Size
- Maximum 500MB per file
- Keep under 5MB for faster loading

### Orientation
- **Portrait only** for iPhone screenshots
- Match device orientation exactly

### Content
- No transparency (solid backgrounds only)
- Must be actual app screenshots (no design mockups)
- No stretched or distorted images
- Status bar can be shown or hidden (preferably hidden)

## Localization

If submitting in multiple languages:
- Create separate screenshot sets for each language
- Translate any text overlays
- Ensure UI text in screenshots matches the target language
- Maintain consistent design across all localized versions

## Testing Before Submission

Before finalizing screenshots:

1. **View on Device**: Check how they look on actual App Store listings
2. **Size Verification**: Confirm exact pixel dimensions
3. **Quality Check**: Zoom in to ensure crisp text and UI elements
4. **Color Accuracy**: Verify dark theme and glassmorphism effects are visible
5. **Content Review**: Ensure no inappropriate or placeholder content
6. **Feature Coverage**: Confirm all key features are represented
7. **Brand Consistency**: All screenshots follow HALO design principles

## App Preview Videos (Optional)

If creating app preview videos:
- **Duration**: 15-30 seconds
- **Orientation**: Portrait
- **Size**: Same as screenshot dimensions
- **Format**: .mov or .mp4
- **No audio required** (but can include ambient sounds)
- **Show actual app functionality** (no animations/transitions that don't exist in app)

### Video Storyboard:
1. Open app (onboarding)
2. Browse discovery feed
3. Tap on a live stream
4. Interact with chat
5. Show safety features
6. End with HALO logo
